Chapter 1
Measurement
Topic-wise quick recap
📏 Introduction · What is Measurement?
  • Measurement = comparing an unknown quantity with a known standard quantity of the same kind.
  • Without measurement, words like "far", "some", "hot" create confusion.
  • Ancient units like foot, cubit varied person to person — standard units were needed.
Formula
Physical Quantity = Numerical Value × Unit
Memory trick

"No Unit = No Meaning"
10 means nothing. 10 m, 10 kg makes sense.

⚖️ 1.1 Unit Systems
  • Old systems created confusion because each country used different units.
  • CGS — Centimetre, Gram, Second — used in small science work.
  • FPS — Foot, Pound, Second — used in Britain/USA earlier.
  • MKS — Metre, Kilogram, Second — basis for SI.
  • SI — improved MKS, accepted worldwide now.
Memory trick

"Come From My School"
CGS, FPS, MKS, SI

🌍 Key Point SI solved confusion in trade and science worldwide.
🌐 1.2 International System of Units (SI)
  • Adopted in 1960 by the 11th General Conference on Weights and Measures.
  • One common system for all countries — no conversion needed.
  • Structure: 7 Base Quantities + 7 Base Units + 2 Supplementary Units.
  • All other units like area (m²), speed (m/s) are derived from these 7.
Memory trick

"Mr. K Sat At Kerala Mall Calmly"
Metre, Kg, Second, Ampere, Kelvin, Mole, Candela

1.2.1 Temperature
  • SI unit = kelvin (K). No degree symbol.
  • 0 K = −273.15°C = Absolute Zero = no heat at all.
  • Used in science and space; daily life uses °C.
K = °C + 273.15
Memory trick

"Kelvin starts from 273"
Body temp 37°C = 310.15 K

1.2.2 Electric Current (I)
  • SI unit = ampere (A), named after André-Marie Ampère.
  • 1 A = 1 coulomb of charge flowing per second.
  • Examples: phone charger ≈ 2 A, house MCB = 5 A or 15 A.
1.2.3 Amount of Substance
  • SI unit = mole (mol).
  • 1 mole = 6.022 × 10²³ particles = Avogadro Number.
  • Like 1 dozen = 12, 1 mole = 6.022 × 10²³ atoms/molecules.
1.2.4 Luminous Intensity
  • SI unit = candela (cd) — measures "brightness" of a light source.
  • 1 cd ≈ brightness of one common wax candle.
  • Examples: mobile torch ≈ 40 cd, car headlight ≈ 1,00,000 cd.
1.2.5 Plane Angle
  • Unit = radian (rad) — a supplementary unit.
  • 1 rad = angle when arc length equals the radius of a circle.
  • Full circle = 2π rad = 360° → 1 rad ≈ 57.3°.
1.2.6 Solid Angle
  • Unit = steradian (sr) — a supplementary unit.
  • 3D angle made at the centre of a sphere, like a cone of torch light.
  • Full sphere = 4π sr.
⏰ 1.3 Clocks
  • Device to measure time intervals accurately.
  • Sundial — uses sun's shadow, fails at night.
  • Water/Sand clock — measures flow of water/sand.
  • Pendulum clock — pendulum swings once per second.
  • Quartz clock — quartz crystal vibrates 32,768 times/sec; used in wall clocks.
  • Atomic clock — counts Cesium-133 atom vibrations; most accurate (error = 1 s in 3 lakh years).
🛰️ Key Point GPS and ISRO satellites use atomic clocks — a 0.001 s error means a 300 km location error.
1.3.1 Types of Clock — By Display
  • Analog — circular dial, 3 hands, angle tells time. Ex: school wall clock.
  • Digital — LED/LCD display, 24-hour or 12-hour format. Ex: mobile, microwave.
1.3.2 Types of Clock — By Mechanism
  • Quartz — battery makes quartz vibrate → very accurate for daily use.
  • Atomic — uses atom vibrations → super accurate for science, GPS.
🎯 1.4 Accuracy in Measurements
  • Accuracy = how close a measured value is to the true value.
  • Least Count = smallest value an instrument can measure — scale 0.1 cm, vernier 0.01 cm, screw gauge 0.001 cm.
  • Zero Error — 0 mark worn out, so all readings go wrong.
  • Parallax Error — seeing the scale from an angle gives a wrong reading; eye must be vertical to the mark.
⚡ To increase accuracy Use a better instrument, take the average of many readings, and avoid parallax.
🔢 1.5 Approximation
  • Writing a number to the nearest 10, 100, or 1000 for fast calculation.
  • Not exact, but useful for estimates — e.g. budget planning, mental maths, big data.
  • Examples: 8274 ≈ 8000, 63 ≈ 60, population ≈ 1 crore.
➗ 1.6 Rounding Off
  • Reducing digits after the decimal to match instrument accuracy.
  • Example: 22 ÷ 7 = 3.142857… — we can't write all the digits, so we round off.
1.6.1 Rules for Rounding Off
  • Rule 1 — dropped digit < 5 → keep last digit same. 7.32 → 7.3
  • Rule 2 — dropped digit > 5 → add 1 to last digit. 7.36 → 7.4
  • Rule 3 — dropped digit = 5 → if last digit even, keep same (7.25 → 7.2); if odd, add 1 (7.35 → 7.4).
Memory trick

"5 wants an Even friend"
5 rounds so the last digit becomes even.

📘 Key Point Always check how many decimal places the question asks for before rounding.

Summary

Plants use sunlight, carbon dioxide and water to produce glucose (food) and release oxygen. This process is called photosynthesis.

✎

Common exam mistake — many students write "plants make oxygen for humans." That's true, but not the main purpose. In exams, write: "The primary purpose of photosynthesis is to produce glucose for the plant. Oxygen is a by-product."

Final challenge — if I cover a plant's leaves with black paper for 3 days, will photosynthesis happen? Why?
No, because the leaves won't receive sunlight.
